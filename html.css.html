How to use HTML!

Hello viewer you may be wondering how is the ashes database created? Well I shall answer your simple question! The ashes database is created with html and css!

What is html you ask?

Html stands for Hypertext Markup Language. Html is a standardized system for tagging text files to achieve font, color, graphic, and hyperlink effects. To put that in simpler terms it's a language that can change everything about itself except for its basic rules.

Who on earth was smart enough to come up with this ingenious language?

In 1980, physicist Tim Berners-Lee, proposed and prototyped ENQUIRE, a system for CERN researchers to use and share documents. In 1989, Berners-Lee wrote a memo proposing an Internet-based hypertext system. Berners-Lee specified HTML and wrote the browser and server software in late 1990. Html is written within tags that could otherwise be used as less than and greater than signs. A few simple rules that follow html are size in text can be modified quickly with style attributes and different “h” tags.

Now tags, you are probably wondering how they work!

Tags need to be opened and closed. This is an open tag, “<”. This is a closing tag “>”. Without these tags most websites would not be able to function.

Now that we have the basic of the basics how do we style our page?

Pages can be styled with style attributes such as color font-size font-family and text-decoration.

Now to list a few of the tags you can use!

  1. p : This tag is used to put words into your site! The p stands for paragraph!
  2. h1 : This is like the “p” tag however this tag makes the text-size very large!
  3. img : This tag is quite long and is used to put images into your site! The full tag is <img src=”(insert url or image)” />
  4. a href : This tag is used to insert links into your site to lead your viewers elsewhere! This tag is also quite long the full tag is <a href=”url”> </a>
